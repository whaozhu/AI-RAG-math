# AI-RAG-math 

## 简介

此 RAG（多轮问答系统）是基于新的数学学术论文（英语）作为外部文档，目标是解决特定的数学问题。

## 目标

- 利用最新的数学研究成果来增强系统的解答能力。
- 实现对特定数学问题的精准解答。

## 功能

- **知识检索**：系统能够从外部数学文献中检索相关信息。
- **多轮对话**：支持与用户进行多轮交互，逐步解决问题。
- **问题解答**：针对特定数学问题提供准确的答案。
